IMPORTANT DESIGN RULE

Keep the exact same colors and visual theme used in the current design. Do not change the color palette. The background, accent color, card styles, and typography must remain exactly the same as the existing UI.

Also maintain the same design language:

- same font
- same rounded cards
- same spacing
- same visual hierarchy

The interface should feel like a modern social community platform combining clubs, events, and a feed.
